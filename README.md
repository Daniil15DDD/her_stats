## ✨ Features

- 🔎 Import profile via in-game UID  
- 📊 DPS Score system (S → SS+)  
- 🛠 Build recommendations (Light Cones, Relics, Planars)  
- 📈 Stat optimization guidance  
- 🏆 Updated tier list  
- 📱 Clean native iOS design  

---

## 🧠 DPS Scoring

Performance is calculated using:

- Main stat optimization  
- Substat weighting  
- Light Cone efficiency  
- Relic synergy  
- Endgame benchmarks  

---

## 🛠 Tech Stack

- Swift  
- SwiftUI  
- REST API integration  
- Local UID storage
